# eth-blchn
### Dependences:
- node.js
- mongodb
- Ganache CLI
- Truffle Suite
### Getting started:
Run emulator blockchain node ethereum:
```
ganache-cli
```
In another shell:
```
git clone https://github.com/rody7val/api_eth_blchn
npm install
cd api_eth_blchn
truffle init
truffle compile
truffle migrate
npm run build
```
